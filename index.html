Optimal Birdwatching in Massachusetts

  • Extracted 8.5GB data, cleaned using 'auk' package in R, and streamlined dataset for preprocessing to analyze with the Prophet model.
  • Optimized model using logistic regression with saturating minimum to stabilize forecasts, cross- validation, grid search, and efficient parameter testing using Pythonâ€™s multiprocessing library. Evaluated model performance with RMSE and MAE metrics.

Fraud Detection Using Scalable Machine Learning

  • Compared accuracy and speed of prediction models ran with Spark's in-memory processing capabilities with prediction models available in Spark MLlib to determine which models are the most efficient in identifying financial fraud.
  • Manually implemented ensemble models with Scikit-Learn (Logistic Regression and Decision Tree). Trained in parallel using Spark.
  • Ran ensemble models and the Spark MLLib models (Decision Tree and Random Forest) on AWS EMR and conducted a comparative analysis of the results, which were stored in AWS S3 buckets.
